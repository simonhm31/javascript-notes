Connecting JS file to HTML

  • Use <script></script> tags to connect JS file to HTML
  • <script src="./file.js"></script>
  • Put script tags before closing </body> tag because it can improve load time

See if JS file is connected:

  • add console.log() statment in JS file
  • Open HTML file in Chrome
  • If connected console.log() statement will appear in JS console
