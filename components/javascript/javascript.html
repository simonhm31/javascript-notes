Variables

Three ways to declare variables in JS:
  1. var name = `john`;
  2. let name = `john`;
  3. const name = `john`;
  • var is not used anymore
  • let variables can be reassigned
  • const variables can never be reassigned

Data Types

  • Data types are attributes associated with a piece of data that tells a computer system how to interpret its value
  • You can find the data type of a variable using typeof
  • Two types of data types: primitive types and reference types
Primitive Data Type Example
string let name = `john`
number let age = 42.3;
Boolean let isGoodAtCoding = true;
undefined let person = undefined;
null let person = null;
