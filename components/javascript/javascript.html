Variables

Three ways to declare variables in JS:
  1. var name = `john`;
  2. let name = `john`;
  3. const name = `john`;
  • var is not used anymore
  • let variables can be reassigned
  • const variables can never be reassigned

Data Types

  • Data types are attributes associated with a piece of data that tells a computer system how to interpret its value
  • You can find the data type of a variable using typeof
  • Two types of data types: primitive types and reference types
Primitive Data Type Example
string let name = `john`
number let age = 42.3;
Boolean let isGoodAtCoding = true;
undefined let person = undefined;
null let person = null;

Operators

Equality Operator Definition
=== strict equality, same value and same type
== same value, not necessarily same type
!= opposite of ==
!== opposite of ===
input output
console.log(5 == '5') true
console.log(5 === '5') false
console.log(5 != '5') false
console.log(5 !== '5') true
Comparison Operators Definition
< less than
> greater than
<= less than or equal to
> greater than or equal to
input output
console.log(5 > 5) false
console.log(5 >= 5) true
console.log(5 < 5) false
console.log(5 <= '5') true
Logical Operators Name Definition
&& Logical And returns true if all operands are true
|| Logical Or returns true if one or more operands is true
! Not changes true to false and false to true
input output
console.log(false && true) false
console.log(7 && 8 && 0) 0 - 7, 8 are truthy, 0 is falsy. && returns first falsy value
console.log(false || true) true
console.log(0 || 7 || 8) 7 - 0 is falsy, 7,8 are truthy. || returns first truthy value
console.log(!true) false
console.log(!(0 || 7 || 8)) false, (0 || 7 || 8) is truthy and !true is false
